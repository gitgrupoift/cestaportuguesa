Manage suppliers, supply orders and stock with WooCommerce.

== Description ==

WooSupply is a free plugin that allows you to create suppliers and supply orders.
Generate your supply orders in PDF and send them to your suppliers.
Receive the ordered products and increase your WooCommerce stocks.
Manage your supplier invoices information and upload documents for further reference.
Access the statistics panel to see your monthly margin based on WooSupply/WooCommerce orders.

== Features ==

* Supplier edition
* Supply orders edition
* PDF Generation
* Stock Entry
* Statistics

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/woosupply` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooSupply screen to configure the plugin
